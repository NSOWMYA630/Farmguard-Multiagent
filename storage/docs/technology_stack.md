# Technology Stack

## Frontend
- React.js
- Next.js
- Tailwind CSS

Purpose:
Used to build the farmer portal, company dashboard, and admin panel.

---

## Backend
- Node.js
- Python
- REST APIs

Purpose:
Handles AI agents, data processing, and system orchestration.

---

## Artificial Intelligence
- Azure AI
- Azure OpenAI
- Computer Vision Models

Purpose:
Analyze farm images, detect trees, and calculate carbon credits.

---

## IoT Devices
- ESP32 Microcontroller
- Soil Moisture Sensor
- Soil pH Sensor
- Weather Sensors

Purpose:
Collect environmental data from farms.

---

## Storage
- Cloud Database
- Azure Blob Storage

Purpose:
Store farm data, carbon calculations, and transaction records.

---

## Blockchain
- Distributed Ledger Technology

Purpose:
Store secure carbon credit transaction records.
