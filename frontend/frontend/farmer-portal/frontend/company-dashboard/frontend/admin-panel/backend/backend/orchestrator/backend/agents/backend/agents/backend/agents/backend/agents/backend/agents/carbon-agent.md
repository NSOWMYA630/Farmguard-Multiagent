# Carbon Analyst Agent

## Purpose
The Carbon Agent calculates carbon credits generated by trees on a farmer’s land.

## Input
- Tree count from Vision Agent
- Tree height
- Tree diameter (DBH)
- Estimated biomass

## Processing Steps

1. Estimate Above Ground Biomass (AGB)
2. Convert biomass into stored carbon
3. Convert carbon into CO₂ equivalent
4. Convert CO₂ into carbon credits

## Carbon Calculation Formula

AGB = 0.0673 × (ρ × DBH² × H)^0.976

Where:
ρ = wood density  
DBH = diameter at breast height  
H = tree height

## Output
- Carbon stored
- CO₂ absorbed
- Carbon credits generated

## Example Output

{
 "carbon_stored": "2.5 tons",
 "co2_equivalent": "9.2 tons",
