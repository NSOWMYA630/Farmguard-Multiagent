# Market Agent

## Purpose
The Market Agent connects farmers with companies that want to purchase carbon credits.

## Input
- Verified carbon credits from Validation Agent
- Company demand for carbon offsets
- Market price of carbon credits

## Processing Steps

1. List available carbon credits from farmers
2. Match companies with available credits
3. Determine carbon credit pricing
4. Facilitate transaction between farmer and company

## Output
- Carbon credit sale confirmation
- Farmer earnings report
- Transaction record
